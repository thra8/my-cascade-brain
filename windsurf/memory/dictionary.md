# 📖 DICTIONNAIRE D'INTENTION AXE

## 🎨 Design & UX
- "Truc stylé / moderne" -> Glassmorphism, flou de mouvement, coins arrondis (2xl)
- "Qui bouge / interactif" -> Framer Motion, hover effects, transitions d'état
- "Propre / minimaliste" -> Atomic Design, espacement conséquent, typographie claire
- "Dark mode / sombre" -> Thème sombre, contrastes élevés, lumières douces
- "Responsive / mobile" -> Mobile-first, breakpoints, touch-friendly

## ⚡ Performance & Vitesse
- "Rapide / fluide" -> Optimisation NumPy, asynchronisme, réduction du bundle
- "Léger / optimisé" -> Tree shaking, lazy loading, code splitting
- "Instantané / immédiat" -> Préchargement, SWR, optimistic updates
- "Scale / gros volume" -> Microservices, cache CDN, pagination infinie

## 🏗️ Architecture & Code
- "Propre / carré" -> Typage fort, Clean Architecture, Atomic Design
- "Solide / robuste" -> Tests unitaires, error boundaries, monitoring
- "Maintenable / évolutif" -> Modularité, documentation, patterns SOLID
- "Production ready" -> CI/CD, monitoring, rollback strategy

## 🔒 Sécurité & Protection
- "Sécurisé / safe" -> Keyring, Secret Shield, audit de dépendances
- "Privé / confidentiel" -> Encryption, GDPR compliance, data minimization
- "Authentification forte" -> 2FA, OAuth2, JWT with refresh tokens
- "Protection anti-attaques" -> CSRF, XSS, rate limiting, input validation

## 📊 Data & Backend
- "Base de données / stockage" -> PostgreSQL, Redis cache, migrations
- "API / services" -> RESTful, GraphQL, rate limiting
- "Analytics / tracking" -> Event tracking, user behavior, metrics
- "Real-time / temps réel" -> WebSockets, Server-Sent Events, streaming

## 🚀 Déploiement & Ops
- "Déployer / mettre en ligne" -> Docker, CI/CD, blue-green deployment
- "Monitoring / observabilité" -> Logs, metrics, health checks
- "Backup / sauvegarde" -> Database backups, disaster recovery
- "Scaling / montée en charge" -> Load balancer, auto-scaling, CDN

## 🧠 IA & Intelligence
- "Intelligent / smart" -> ML models, recommendation engine
- "Auto-complétion / suggestions" -> Fuzzy search, predictive text
- "Analyse / insights" -> Data visualization, dashboards
- "Personnalisation" -> User preferences, adaptive UI

## 💡 Business & Features
- "E-commerce / boutique" -> Cart, checkout, inventory management
- "Social / communauté" -> User profiles, feeds, notifications
- "Productivité / outils" -> Task management, calendars, collaboration
- "Contenu / CMS" -> Rich text editor, media management, SEO
